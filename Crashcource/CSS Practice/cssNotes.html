Styling CSS elements

You have multiple ways to reference an HTML element to style it
  • Tag name
    • <div></div>
    div{color: red;}
  • Class
    • <div class="text"></div>
    .text{color: red;}
  • Id
    • <div id="test"></div>
    #test{color: red;}

What does it consist of?
  • content

Display

Two CSS display values
  • Block
    • Takes 100% of the width and starts on a new line
    e.g <p> <div> tags
  • inline
    • Doesn't allow you to set a width or height for elements & starts on the same line
    e.g <span> <img> tag

Position

CSS position values (common)
  • Static (default)
    • Follows normal flow of the page
  • Relative
    • Follows normal flow of the page
    Lets you position using top,left,bottom,right
  • absolute
    • Does not follow the normal flow of the page
    Lets you position using top,left,bottom,right
    Position is based off closest 'position: realtive' parent
  • fixed
    • Sticks elements to a permanent location on the page
    Uses top, left, bottom, right
  • sticky
    • sticks element to a permanent location on the page once it hits a threhold

BEM

Block Element Modifier
  • B [block] is the block component
  • E [element] is dependent on the block component
  • M [modifier] is used to change the style of the block
